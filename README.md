# ⚡ Koy Programming Language

**Koy** — это реактивный скриптовый язык программирования, разработанный на Dart. Язык поддерживает сигналы для удобного отслеживания изменений данных и поставляется с готовым CLI-интерпретатором.

---

## 🚀 Быстрый старт

### 1. Установка
Скачайте последний выпуск инсталлятора **`KoySetup.exe`** из раздела Releases и запустите его. 
Установщик автоматически скопирует необходимые файлы и добавит **Koy** в системную переменную `PATH`.

> 💡 *После установки обязательно перезапустите открытые окна терминала!*
