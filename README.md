# ⚡ Koy Programming Language

**Koy** — это реактивный скриптовый язык программирования, разработанный на Dart. Язык поддерживает сигналы для удобного отслеживания изменений данных и поставляется с готовым CLI-интерпретатором.

---

## 🚀 Быстрый старт

### 1. Установка
Скачайте последний выпуск инсталлятора **`KoySetup.exe`** из раздела Releases и запустите его. 
Установщик автоматически скопирует необходимые файлы и добавит **Koy** в системную переменную `PATH`.

><!TIP>
> *После установки обязательно перезапустите открытые окна терминала!*

---

## 💻 Использование

### Проверка установки
Откройте любой терминал (CMD, PowerShell, Git Bash) и выполните:
```cmd
koy --version

```

### Запуск `.koy` скриптов

Для выполнения файла с кодом передайте путь к нему в команду `koy`:

```cmd
koy script.koy

```

---

## 📝 Пример кода на Koy

Создайте файл `main.koy` и добавьте следующий код:

```Koy
// Вывод сообщений в консоль
Logger.log("Привет мир из Koy!");

// Объявление реактивного сигнала
signal counter = 1;

// Подписка на изменения сигнала
counter.listen((val) {
  Logger.log("Значение counter изменилось на: " + val);
});

// Изменение значения триггерит слушатель
counter.value = 100;

```

Запустите скрипт:

```cmd
koy main.koy

```

## 🛠 Сборка из исходников

Если вы хотите собрать интерпретатор самостоятельно:

1. Убедитесь, что у вас установлен [Dart SDK](https://dart.dev/get-dart).
2. Клонируйте репозиторий:
```cmd
git clone [https://github.com/asanalinskiy/koy.git](https://github.com/asanalinskiy/koy.git)
cd koy
```


3. Скомпилируйте проект в `.exe`:
```cmd
dart compile exe bin/main.dart -o koy.exe

```
